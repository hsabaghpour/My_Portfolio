LTE Handover Optimization: Graph ML & GNN Approache

This project focus on optimizing handover success rates in an LTE network by modeling eNodeBs as nodes and NBR relations as edges. The first approach uses Graph-Based Machine Learning, where a logistic regression model is applied to predict and adjust NBR relations based on features such as node degree and handover success rate. The second approach utilizes a Graph Neural Network (GNN), which learns node and edge embeddings to predict successful handovers, using graph convolutions to aggregate information from neighboring nodes. Both methods aim to enhance network performance by eliminating overshooting NBRs and adding missing connections.

CNN on the MNIST Dataset with PyTorch


a trial on the MNIST dataset using PyTorch and Convolutional Neural Networks (CNNs). Convolutional layers scan images for edges and patterns, ReLU activation functions help recognize complex patterns, and pooling layers summarize features for efficiency. Fully connected layers make the final decisions, with the output layer predicting digits like "9." CNNs are efficient due to local connectivity and shared weights, train faster with fewer parameters, and recognize objects regardless of position. They build knowledge hierarchically and are optimized for modern hardware. My trial visualized convolutional and pooling layers, showing max pooling retains key features while average pooling smooths representation. CNNs are essential for facial recognition, self-driving cars, and medical image analysis, demonstrating their powerful capabilities in image processing.
