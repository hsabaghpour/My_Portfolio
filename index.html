Work Experiences and Projects

Support Vector Machine trial on IRIS data set
2024

Exploring Support Vector Machines (SVMs) with the Iris dataset! SVMs shine in classification, regression, and outlier detection, particularly with smaller datasets like Iris. The Linear SVM method seeks the widest street (large margin) to separate classes using support vectors, emphasizing the importance of scaled features. Soft Margin, on the other hand, offers flexibility by balancing margin width and violations, controlled by the CCC parameter. My recent work with the Iris dataset using SVMs is available on my GitHub page
